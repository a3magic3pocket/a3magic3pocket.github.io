    • Githubмқҙ нҳёмҠӨнҢ… м„ңлІ„ м—ӯн• мқ„ н•ҙмӨҢмңјлЎң мҡ°лҰ¬лҠ” Jekyll нҢҢмқјл§Ң мғқм„ұн•ҳм—¬ м—…лЎңл“ңн•ҳл©ҙ
      мһҗлҸҷмңјлЎң Github м„ңлІ„м—җм„ң мһҗлҸҷмңјлЎң л№Ңл“ңн•ҳм—¬ нҳёмҠӨнҢ…н•ҙмӨҖлӢӨ.
  • Jekyll мқҙлһҖ?
    • Ruby м–ём–ҙлЎң л§Ңл“Өм–ҙм§„ м •м Ғ мӮ¬мқҙнҠё мғқм„ұкё°

    • Jkyll кіөмӢқ лҸ„нҒҗлЁјнҠём—җм„ңмқҳ Jekyll м •мқҳ
      Jekyll мқҖ м •м Ғ мӮ¬мқҙнҠё мғқм„ұкё°мһ…лӢҲлӢӨ.
      лӢ№мӢ мқҙ мҰҗкІЁ мӮ¬мҡ©н•ҳлҠ” л§ҲнҒ¬м—… м–ём–ҙлЎң мһ‘м„ұлҗң н…ҚмҠӨнҠёлҘј Jekyll м—җ л„ҳкІЁмЈјл©ҙ
      л Ҳмқҙм•„мӣғмқ„ мӮ¬мҡ©н•ҙ м •м Ғ мӣ№мӮ¬мқҙнҠёлҘј мғқм„ұн•©лӢҲлӢӨ.
      мӮ¬мқҙнҠё URL мқҳ нҳ•мӢқмқҙлӮҳ м–ҙл–Ө лҚ°мқҙн„°лҘј мӮ¬мқҙнҠём—җ н‘ңмӢңн•  кІғмқём§Җ л“ұ,
      м—¬лҹ¬ лҸҷмһ‘мқ„ мЎ°м •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • Jekyll кіөмӢқ лҸ„нҒҗлЁјнҠё

лҢҖмғҒ лҸ…мһҗ

  • Github кі„м •мқҙ мһҲкі  мӮ¬мқҙнҠём—җ лЎңк·ёмқён•ң мғҒнғң, github нҡҢмӣҗк°Җмһ…
  • docker м„Өм№ҳ лҗҳм–ҙмһҲлҠ” мғҒнғң, docker for windows м„Өм№ҳ
  • git for windows м„Өм№ҳ лҗҳм–ҙмһҲлҠ” мғҒнғң, git for windows м„Өм№ҳ
  • vscodeк°Җ м„Өм№ҳлҗҳм–ҙ мһҲлҠ” мғҒнғң(лӢӨлҘё м—җл””н„°лҸ„ мғҒкҙҖм—ҶлӢӨ), vscode м„Өм№ҳ

Jekyll н…Ңл§Ҳ м„ нғқ л°Ҹ нҸ¬нҒ¬

  • м„ӨлӘ…
    • Jekyll кіөмӢқ лҸ„нҒҗлЁјнҠёлҘј ліҙкі  лӮҙк°Җ мӣҗн•ҳлҠ” мӣ№ мӮ¬мқҙнҠёлҘј л§Ңл“Ө мҲҳ мһҲм§Җл§Ң,
      лӢӨлҘё нӣҢлҘӯн•ң к°ңл°ңмһҗк°Җ л§Ңл“Өм–ҙ л‘” н…Ңл§ҲлҘј мӮ¬мҡ©н•  мҲҳлҸ„ мһҲлӢӨ.
  • мӢӨн–үл°©лІ•
    • мӣ№ лёҢлқјмҡ°м ҖлҘј мјңкі  Jekyll н…Ңл§Ҳ кІҖмғү мӮ¬мқҙнҠёлЎң мқҙлҸҷн•ңлӢӨ.
    • мӣҗн•ҳлҠ” н…Ңл§ҲлҘј кІҖмғүн•ҳкі  м„ нғқн•ңлӢӨ(лӮҳлҠ” ChirpyлҘј м„ нғқн–ҲлӢӨ).
    • м—¬кё°м„ң нҷҲнҺҳмқҙм§ҖлҘј лҲ„лҘёлӢӨ.
    • Jekyll themeмқҳ github repositoryк°Җ лңЁл©ҙ forkлҘј лҲ„лҘёлӢӨ.
    • fork лң¬ repositoryм—җм„ң settingsлҘј лҲ„лҘёлӢӨ.
    • reposiory мқҙлҰ„мқ„ [github user name].github.ioлЎң л°”кҫјлӢӨ.
  • нҷ•мқё
